## ğŸŒ How It Works

### 1. **Dynamic Content Loading**
- Project data is stored in `data/projects.json` (supports both Italian and English)
- JavaScript dynamically renders projects on `projects.html`
- Case study details are loaded from the JSON data

### 2. **Navigation Structure**
- **Homepage** (`index.html`): Hero section, services overview, featured projects, contact CTA
- **Projects Page** (`projects.html`): Full project gallery with filtering and details
- **Case Studies** (`cases/case-study.html`): In-depth project analysis with challenge, solution, results, and key learnings

- Both Italian (`it`) and English (`en`) content in `projects.json`
- Language switching capability for seamless user experience

- Mobile-first approach with CSS media queries
- Touch-friendly navigation and interactive elements
- Optimized performance across all devices
